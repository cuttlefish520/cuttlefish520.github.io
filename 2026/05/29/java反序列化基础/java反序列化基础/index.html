1.java反序列化

java序列化是把java对象转换成字节序列的过程，用来储存数据，字节数据可以储存在文件或内存中。

java反序列化就是通过字节数据恢复储存的对象。

2.实现方法

序列化：ObjectOutputStream类中的writeObject()方法

反序列化：ObjectInputStream类中的readObject()方法

举一个列子

下面解释

Serializable接口

在定义类是需要继承Serializable接口，在能对类的对象进行序列化。

FileOutputStream：文件字节输出流

FileOutputStream是java.io模块里的一个控制字节输出流的类，简单说就是字节数据输出到哪。

FileOutputStream会把字节数据写入到磁盘文件里，出来FileOutputStream还有一个常用的ByteArrayOutputStream

ByteArrayOutputStream：字节数组输出流

将内存中的字节数据，写入到 JVM 内部的一个动态字节数组中。所有数据都只存在于内存里，不会写到硬盘。当程序结束是数据也会消失，与FileOutputStream不同不会永久保存在磁盘中
